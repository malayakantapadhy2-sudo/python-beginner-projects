# HTML Calculator

This project is a simple and realistic calculator built using HTML, CSS, and JavaScript.  
It performs basic arithmetic operations and runs directly in a web browser.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Percentage
- Clear button
- User-friendly calculator interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run the Project
1. Download or clone this repository.
2. Open the file `calculator.html`.
3. Double-click the file or open it in any web browser.

The calculator will run directly in the browser.
